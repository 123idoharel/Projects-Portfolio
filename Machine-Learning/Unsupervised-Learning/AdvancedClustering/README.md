# Unsupervised Learning – Clustering & Kaggle Submission

## Overview
This project focuses on applying unsupervised learning techniques to discover hidden patterns in unlabeled data and generate structured insights.  
The work includes exploratory analysis, clustering model development, and preparation of a submission for a Kaggle evaluation task.

The project demonstrates a full unsupervised learning workflow: data preprocessing, model selection, evaluation, and practical deployment.

---

## Objectives
- Analyze the structure of unlabeled data
- Identify natural groupings using clustering methods
- Evaluate cluster quality using quantitative metrics
- Generate predictions for external evaluation (Kaggle)

---

## Methodology

### Data Preprocessing
- Handling missing values and outliers
- Feature scaling and normalization
- Exploratory Data Analysis (EDA)

### Clustering Techniques
- K-Means clustering
- Selection of optimal number of clusters using:
  - Elbow Method
  - Silhouette Score

### Model Evaluation
- Cluster cohesion and separation analysis
- Visualization of cluster distribution
- Interpretation of cluster characteristics

### Kaggle Submission
- Final clustering configuration applied to test data
- Exported predictions for external scoring
